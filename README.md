# play_audio

为了解决浏览器无操作时限制播放声音的问题，用 flutter 方案编写的 windows 声音播放插件，绑定 localhost:8086 端口，浏览器调用接口时播放声音，支持开机自启

flutter build windows 生成 windows 打包资源后，配合 inno 可以生成 windows 安装包
